# Hiredge

Hiredge is a full-stack job application management workspace that helps users track applications, deadlines, interview stages, offers, and resume insights from one place.

## Live Demo

[Open Hiredge](https://hiredge-xi.vercel.app)

## Overview

Job seekers often manage applications across LinkedIn, email, spreadsheets, and company portals. Hiredge brings that process into one focused dashboard where users can securely track every opportunity from saved to offer.

## Features

- Secure sign up, sign in, and logout
- Private user-specific workspace
- Add, edit, and delete job applications
- Track application status
- Search, filter, and sort applications
- Deadline and applied-date tracking
- Dashboard analytics and funnel overview
- Recent activity and next-action tracking
- Resume insights with transparent keyword comparison
- Responsive desktop and mobile interface
- Accessible forms and controls
- Dark professional SaaS interface

## Application Statuses

- Saved
- Applied
- Interview
- Offer
- Accepted
- Rejected

## Resume Insights

The Resume Insights feature compares resume text with a job description using transparent keyword matching.

It highlights:

- Matched keywords
- Missing keywords
- Relevant technical terms
- Practical improvement suggestions

The generated score is an assistive estimate and not a hiring prediction.
